# 系统架构
---

## 整体结构

<Image src={architectureImage} alt="architecture" className="rounded-lg" placeholder="empty"/>

### 1. 管理面板

- 前端界面：提供统一的 VPS 管理视图
- API 集成层：对接各大主机商 API
- 数据处理层：聚合和分析来自不同来源的数据

### 2. Rust 探针系统

- 轻量级服务器端组件
- 通过 WebSockets 协议进行实时数据传输
- 低资源占用，高性能监控

### 3. 通信架构

- WebSockets 作为核心通信协议
- 双向实时数据流
- 支持断线重连和数据缓存机制
