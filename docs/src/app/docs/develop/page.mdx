# 开发

## 设置数据库

<DatabasePassword>

```bash
docker run -d --name vmboard-db -p 5432:5432 \ 
  -v vmboard-db-data:/home/postgres/pgdata/data \ 
  -e POSTGRES_PASSWORD=postgres \
  timescale/timescaledb-ha:pg17
```

</DatabasePassword>


### 克隆仓库
```bash
git clone https://github.com/achat/vmboard.git && cd vmboard
```

### 安装依赖
```bash
pnpm install
```

### 同步数据库
```bash
pnpm prisma:migrate
```

### 启动服务
```bash
pnpm dev
```

服务将会在 `http://localhost:3000` 启动。
